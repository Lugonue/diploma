

## Запуск
1) Если нет менеджера пакетов yarn ставим
```
npm i -g yarn
```
2) Подтягиваем зависимости
```
yarn
```
3) Запускаем дев версию (http://localhost:3000/)
```
yarn dev
```
4) Билд

```
yarn build
```
